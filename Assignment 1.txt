Q1. How can you output “Physics” and “Wallah” in two different lines in C++?
Answer - 
“Physics” and “Wallah” in two different lines, we can write using endl and \n.
For example - 


#include<iostream>
using namespace std;
int main(){
        cout<<”Physics”<<endl;
        cout<<”Wallah”;
        return 0;
}


Q2. Write a program to print 10 using 2 positive numbers less than 6 in C++ as output.
Answer -
#include<iostream>
using namespace std;
int main(){
        int a = 4;
        int b = 2;
        int c = (a*b)+b;

}

Q5.Write a program to find the circumference of a circle with radius 10 in C++.
Answer -
#include<iostream>
using namespace std;
int main(){
        float pi = 3.14;
        float radius = 5;
        // circumference of circle
        float circumference = 2*pi*radius;
        cout<<circumference;
}
